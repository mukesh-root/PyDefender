## Features
- Signature-based detection
- Behavioral analysis
- Entropy calculation
- Comprehensive reporting

## Installation
```bash
pip install git+https://github.com/mukesh-root/PyDefender.git
```

## Usage
```bash
# Quick system scan
pydefender --quick

# Full scan with report
pydefender /path/to/scan -o report.json
```
